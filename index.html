Value Sets for EQ5D5L countries in ALS CarE

Finding out how to score EQ5D5L for each country is a pain in the teeth and you have to root through loads of papers and suplementary materials etc. so I am documenting them here for posterity, or at least the scoring methods required for the sites in the ALS CarE study. The formula, or code, for each region is listed below

  • Ireland (Hobbins et al, 2018):
    V(X) = 1 - 0.063MO2 - 0.097M03 - 0.215M04 - 0.344M05 - 0.055SC2 - 0.088SC3 - 0.229SC4 - 0.287SC5 - .049UA2 - 0.072UA3 - 0.154UA4 - 0.187UA5 - 0.068PD2 - 0.093PD3 - 0.373PD4 - 0.514PD5 - 0.080AD2 - 0.202AD3 - 0.535AD4 - 0.646AD5
  • England:
    Use the following command in R: valueEQ5D::map_5Lto3L().
    You'll have to do it this way which maps the old index values from EQ5D3L to EQ5D5L for now because the NICE have found a problem with the English value sets for EQ5D5L (Van Hout et al, 2012).
  • Italy (based on Finch et al (2022)):
    V(X) = 1 - 0.051MO2 - 0.064M03 - 0.244M04 - 0.329M05 - 0.046SC2 - 0.056SC3 - 0.216SC4 - 0.257SC5 - 0.050 UA2 - 0.064UA3 - 0.225UA4 - 0.255UA5 - 0.047PD2 - 0.088PD3 - 0.353PD4 - 0.408PD5 - 0.044AD2 - 0.109AD3 - 0.318AD4 - 0.322AD5
